The Beatles Member Classifier
====
The Beatlesのメンバーを判別出来る画像分類AIです。

## Description
初めて機械学習のアプリを作成しました。
画像をアップロードすると、顔検出し、どのメンバーか判別します。
精度は90％ほど。
